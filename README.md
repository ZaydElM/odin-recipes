# Odin Recipes

A simple multi-page recipe website built as part of The Odin Project curriculum. This project focuses on practicing core HTML concepts by creating a structured and navigable collection of recipes from different cuisines.

## 🔗 Live Demo
https://zaydelm.github.io/odin-recipes/

## 📌 Overview
The site presents a selection of recipes, each on its own page, with clear navigation between them. The goal was to build a solid foundation in HTML by organizing content in a meaningful and accessible way.

## 🛠️ Built With
- HTML5
- Semantic markup

## ✨ Features
- Multi-page layout with linked recipe pages  
- Use of semantic HTML elements for better structure  
- Organized content using headings, lists, and paragraphs  
- Images to enhance visual presentation  
- Navigation using both relative and absolute links  

## 📚 What I Learned
- Structuring a basic website using HTML  
- Creating and linking multiple pages  
- Writing clean and readable markup  
- Organizing content with lists and semantic elements  

## 🚀 Future Improvements
- Add CSS styling for layout and design  
- Improve responsiveness for mobile devices  
- Include more recipes and categories
